Apps Portfolio

Notify Plus

Notify Plus is a feature-rich tool box that makes your Android experience even more fun! The app provides you with 2 main tools:

  1. A sticky notification containing 8 modifiable shortcuts to user's favorite apps.
  2. A floating widget which can work as an equivalent of floating control center in iOS.

Apart from all the fun, the most important experiences I took from this project were these:

  1. Git Version Control
  2. RemoteViews in Android
  3. Sticky notifications in Android
  4. GUI control via services
  5. Usage of 3rd party libraries via Gradle dependencies

The installable APK of the app is available here.

Room Demo

Historically, Android apps used SQLite API for creating a local relational database. While that worked fine, the modern Room persistence library offers compile-tile SQL query verification and convenience annotations to reduce code lines.
"Room Demo" is my template app for creating a single-table relational database storing subscribers. Source code is available here.
Technologies used:

  • Room persistence library
  • MVVM architecture
  • ViewModel
  • DataBinding
  • LiveData
  • kotlin coroutines

Retrofit Demo

Retrofit library is the most common choice of Java developers for connecting to a RESTFul API in the back end.
"Retrofit Demo" is my template app for reading JSON file from server and then present it to the user with the help of LiveData and kotlin Coroutine libraries. The JSON data is parsed into kotlin data classes with GSON converter library; furthermore, OkHttp logging interceptor was used for logging network interactions. Source code is available here.
Technologies used:

  • Retrofit
  • GSON
  • Kotlin Coroutines
  • LiveData
  • OkHttp3 logging interceptor

RxJava Demo

Reactive programming paradigm is the best answer for the problem of handling realtime streams of data and state change propagation in an asynchronous fashion; and ReactiveX library helps in achieving reactiveness in a langauge independent fashion. Consequently, RxJava and RxAndroid are de facto path for performing reactive, event-driven apps in native Android.
"RxJava Demo" is my template app for easily implementing reactive patterns of RxJava in an Android app. Source code is available here.
